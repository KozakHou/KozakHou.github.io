# PoHan (Kozak) Hou

Personal academic site: [kozakhou.github.io](https://kozakhou.github.io)

Built with [Jekyll](https://jekyllrb.com/) and a customized [al-folio](https://github.com/alshedivat/al-folio)-based editorial theme.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

Or with Docker:

```bash
docker compose up
```

Then open `http://127.0.0.1:8080`.
